🚀 Quick Setup

1
Pull TTS Docker Image: docker pull ghcr.io/remsky/kokoro-fastapi-cpu:latest
2
Start TTS Service: docker run -d --name local-tts-cpu -p 8880:8880 ghcr.io/remsky/kokoro-fastapi-cpu:latest
3
Set API URL: Use http://localhost:8880
4
Click "Read" to start using the extension!

📖 How to Use

📄
Read Full Article: Click the extension icon and press "Read"
✂️
Read Selected Text: Select text on the page, right-click, and choose "Read selected text"
⏯️
Control Playback: Use Pause/Stop buttons or keyboard shortcuts (Space, Esc)
🎛️
Adjust Settings: Change voice, speed, and audio format in the Settings tab

🔧 Troubleshooting

Can't connect to API?
  • Check if Docker container is running
  • Verify API URL is correct
  • Try restarting the container
No audio playing?
  • Check browser audio settings
  • Click on the page first
  • Try refreshing the page
